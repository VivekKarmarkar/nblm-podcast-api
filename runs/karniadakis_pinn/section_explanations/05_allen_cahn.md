# Section 3.1.1 — Allen-Cahn Equation (pages 17-18)

## Why this example exists
This is the second discrete-time benchmark — included to demonstrate the discrete-time approach can handle a DIFFERENT FLAVOR of nonlinearity than Burgers had. Burgers had a quadratic nonlinearity (u·u_x). Allen-Cahn has a CUBIC nonlinearity (5u³ − 5u). Different shape, different challenges.

## The physics
The Allen-Cahn equation comes from reaction-diffusion systems and is used to describe phase separation in multi-component alloy systems — order-disorder transitions, where a uniform mixture spontaneously separates into distinct domains. It's a classic toy model for nonlinear pattern formation.

The instance solved:
> u_t − 0.0001·u_xx + 5u³ − 5u = 0,  x ∈ [−1, 1],  t ∈ [0, 1]
> u(0, x) = x²·cos(πx)
> u(t, −1) = u(t, 1), u_x(t, −1) = u_x(t, 1)  [periodic]

The nonlinear operator is N[u_{n+cj}] = −0.0001·u_xx^{n+cj} + 5·(u_{n+cj})³ − 5·u_{n+cj}. Apart from the periodic boundary handling, the structure mirrors section 3.1.

## Reference solution
Generated by integrating equation (13) from a known initial condition u(0, x) = x²·cos(πx) up to t = 1.0 using the Chebfun package with 512 Fourier modes and a fourth-order explicit Runge-Kutta scheme with a tiny time step Δt = 10⁻⁵. This generates the "ground truth" that the PINN's prediction will be checked against.

## The test
- Nn = 200 initial data points, randomly sub-sampled from the exact solution at t = 0.1
- Goal: predict the solution at t = 0.9 in a SINGLE time step of size Δt = 0.8
- Discrete-time PINN: 4 hidden layers, 200 neurons per layer
- Output layer predicts 101 quantities corresponding to q = 100 Runge-Kutta stages plus u_{n+1}(x)

## Result
Relative L2 error of 6.99 × 10⁻³ at t = 0.9.

The load-bearing observation from the paper:
> "Despite the complex dynamics leading to a solution with two sharp internal layers, we are able to obtain an accurate prediction of the solution at t = 0.9 using only a small number of scattered measurements at t = 0.1."

Allen-Cahn famously develops sharp internal layers (the fronts between separated phases). The PINN resolves them from 200 sparse points without any classical spatial discretization, in one giant time step.

## Why this matters for the broader story
The Burgers' rerun in section 3.1 showed discrete-time PINNs work on a quadratic nonlinearity. The Allen-Cahn rerun here shows they work on a cubic one. Together they demonstrate that the discrete-time framework isn't tied to any particular flavor of PDE — the Runge-Kutta backbone + neural network state representation generalizes to whatever nonlinear operator N[u] you plug in. Same recipe, different physics.
